#  Portfolio Personnel - Thomas ISSOKO

Deployment  📌-   https://mon-portfolio-kohl.vercel.app/

## 📌 Description

Ce projet est un portfolio personnel complet et professionnel, conçu pour présenter mon parcours, mes compétences techniques et mes réalisations en tant que développeur web Full Stack en devenir. Il s'agit d'une vitrine interactive et moderne qui regroupe l'ensemble de mes projets académiques et personnels réalisés dans le cadre de ma formation à l'Akieni Academy. Ce portfolio a pour objectif de démontrer ma maîtrise des technologies front-end fondamentales, ma capacité à créer des interfaces utilisateur responsives et accessibles, ainsi que ma progression dans le domaine du développement web. Il constitue un outil de présentation essentiel pour mon insertion professionnelle et me permet de partager mon travail avec des recruteurs, des collaborateurs potentiels ou toute personne intéressée par mon profil. Le site est entièrement responsive, accessible sur tous les appareils, et intègre des fonctionnalités modernes telles que le mode sombre et des animations au scroll pour offrir une expérience utilisateur fluide et agréable.

## 🎯 Objectifs du projet

Ce projet a été réalisé avec plusieurs objectifs pédagogiques et professionnels clairs. Tout d'abord, il s'agit de mettre en pratique l'ensemble des compétences acquises au cours de ma formation Full Stack, en particulier en HTML5, CSS3, et Git/GitHub. Le portfolio me permet de démontrer ma capacité à créer un site web complet de A à Z, en respectant les bonnes pratiques du développement front-end. Ensuite, ce projet vise à développer une identité visuelle professionnelle et personnelle, en créant un design unique et cohérent qui reflète ma personnalité et mon approche du développement. J'ai également travaillé sur l'accessibilité et les performances, en m'assurant que le site soit utilisable par tous et qu'il se charge rapidement. Enfin, ce portfolio constitue un livrable concret pour ma formation à l'Akieni Academy, démontrant ma capacité à mener un projet complexe de bout en bout, de la conception initiale au déploiement en ligne, en passant par la gestion de versions avec Git et le déploiement sur GitHub Pages.

## 🛠️ Technologies utilisées

- **HTML5** : Structure sémantique du site, respect des normes d'accessibilité WCAG.
- **CSS3** : Mise en page et stylisation complète avec variables CSS, Flexbox, CSS Grid, animations et transitions.
- **JavaScript (Vanilla)** : Interactivités telles que le menu burger mobile, le mode sombre avec sauvegarde locale, et les animations au scroll.
- **Git & GitHub** : Gestion de versions avec 20+ commits, hébergement du code source et déploiement via GitHub Pages.
- **Google Fonts** : Typographie moderne avec la police Poppins.
- **Font Awesome** : Icônes pour les réseaux sociaux et les éléments d'interface.
- **VS Code** : Environnement de développement avec Live Server pour le rendu en temps réel.

##  Fonctionnalités

Le portfolio intègre un ensemble complet de fonctionnalités conçues pour offrir une expérience utilisateur professionnelle et engageante. Une barre de navigation fixe permet d'accéder rapidement aux différentes sections du site, avec un menu burger adaptatif pour les appareils mobiles. La section Hero présente une photo de profil, un titre accrocheur et un sous-titre qui résume mon identité professionnelle, accompagné d'un appel à l'action invitant à découvrir mes projets. La section À propos détaille mon parcours, ma formation actuelle à l'Akieni Academy et mes aspirations professionnelles. La section Compétences affiche mes niveaux de maîtrise technique à travers des barres de progression animées, couvrant HTML5, CSS3, JavaScript, les bases de données, Git et le responsive design. La galerie de projets présente six réalisations concrètes, chacune avec une image miniature, un titre, une description courte, des tags technologiques, des liens vers GitHub, et une description longue apparaissant au survol. La timeline d'expérience retrace mon évolution dans le domaine de l'informatique sur les deux dernières années. Un formulaire de contact permet aux visiteurs de m'envoyer des messages, accompagné de mes coordonnées et de liens vers mes profils sociaux. Le footer contient les mentions légales et les réseaux sociaux. Des fonctionnalités bonus incluent un mode sombre avec persistance locale, des animations au scroll qui font apparaître les sections en fondu, et un design entièrement responsive avec trois points de rupture adaptés aux mobiles, tablettes et grands écrans.

## ⚙️ Installation et utilisation

Pour installer et utiliser ce portfolio localement, commencez par cloner le dépôt GitHub sur votre machine en utilisant la commande `git clone https://github.com/thomas-issoko/mon-portfolio.git`. Une fois le dépôt cloné, ouvrez le dossier `mon-portfolio` dans Visual Studio Code ou votre éditeur préféré. Aucune dépendance n'est requise car le projet utilise uniquement des technologies front-end pures (HTML, CSS, JavaScript). Pour visualiser le site, ouvrez simplement le fichier `index.html` dans votre navigateur. Pour une expérience de développement optimale, il est recommandé d'utiliser l'extension Live Server dans VS Code, qui permet d'obtenir un rechargement automatique à chaque modification. Si vous souhaitez personnaliser le contenu, modifiez les sections correspondantes dans `index.html` (texte, images, liens, compétences, projets) et ajustez le style dans `css/style.css`. Pour ajouter vos propres images, placez-les dans le dossier `images` en respectant les noms de fichiers attendus (profile.jpg, about.jpg, project1.jpg, etc.). Le site est conçu pour fonctionner sans connexion internet une fois les polices Google Fonts et Font Awesome chargées initialement.

##  Commandes utiles

```
# Cloner le dépôt
git clone https://github.com/thomas-issoko/mon-portfolio.git

# Initialiser Git (si cloné manuellement)
git init

# Ajouter tous les fichiers au suivi de version
git add .

# Créer un commit avec un message descriptif
git commit -m "Message du commit"

# Lier le dépôt local au dépôt distant GitHub
git remote add origin https://github.com/thomas-issoko/mon-portfolio.git

# Pousser les modifications vers GitHub
git push -u origin main

# Voir l'historique des commits
git log --oneline

# Ouvrir le projet avec Live Server (VS Code)
Clique droit sur index.html → "Open with Live Server"

# Visualiser le site dans le navigateur
Ouvrir index.html directement dans le navigateur
```

## 👀 Aperçu

Le portfolio présente un design moderne et épuré avec une identité visuelle forte basée sur une palette de deux couleurs principales : un bleu nuit profond (#0a192f) et un bleu/vert néon (#64ffda) qui créent un contraste élégant et professionnel. La typographie Poppins apporte une touche contemporaine et une excellente lisibilité. La page débute par une section Hero accueillante avec une photo de profil circulaire, un titre dynamique et un sous-titre qui résume l'identité professionnelle. La navigation est intuitive avec une barre fixe contenant les liens vers les différentes sections et un bouton de basculement vers le mode sombre. Chaque section est clairement délimitée et structurée, avec des titres soulignés par un trait décoratif vert néon. Les compétences sont représentées par des barres de progression animées, tandis que les projets sont présentés sous forme de cartes modernes avec des images, des tags technologiques et des liens d'accès. La timeline d'expérience offre une visualisation chronologique du parcours, et le formulaire de contact est intégré avec des icônes et des liens vers les réseaux sociaux. Le design est entièrement responsive, s'adaptant parfaitement aux mobiles, tablettes et grands écrans, avec des animations subtiles au scroll qui améliorent l'expérience utilisateur. Le mode sombre, activable par un simple clic, offre une alternative visuelle confortable pour la navigation nocturne. L'ensemble du site respire le professionnalisme, la créativité et l'attention portée aux détails, reflétant les compétences et la personnalité de son auteur.

## 🔧 Améliorations possibles

Plusieurs améliorations pourraient être apportées à ce portfolio pour le rendre encore plus complet et interactif. Tout d'abord, l'ajout d'un formulaire de contact fonctionnel avec un backend (PHP, Node.js ou un service comme Formspree) permettrait aux visiteurs d'envoyer des messages directement. L'intégration d'un système de filtrage des projets par technologie améliorerait la navigation dans la galerie. L'ajout d'une section blog permettrait de partager des articles techniques et des retours d'expérience. La mise en place d'un système de newsletter pourrait fidéliser les visiteurs. L'optimisation des images avec des formats modernes comme WebP améliorerait les performances de chargement. L'ajout d'un système de multilinguisme (français/anglais) élargirait l'audience du portfolio. L'utilisation d'un framework JavaScript comme React ou Vue pourrait être envisagée pour une future refonte, bien que le vanilla JS soit parfaitement adapté pour ce projet. L'intégration d'animations plus avancées avec des librairies comme GSAP pourrait apporter une touche encore plus professionnelle. Enfin, l'optimisation du référencement (SEO) avec des balises meta enrichies et un plan du site permettrait d'améliorer la visibilité sur les moteurs de recherche.
